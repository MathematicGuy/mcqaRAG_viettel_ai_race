## C ử a s ổ ch ế độ và tr ạ ng thái

Cho bi ế t h ệ th ố ng nào đang ho ạ t độ ng, tr ạ ng thái c ủ a h ệ th ống đó và có bấ t k ỳ báo độ ng nào hay không.

## 2. Các bi ểu tượ ng chuy ển độ ng hi ệ n ho ạ t

Nh ữ ng bi ểu tượ ng này hi ể n th ị phía trên thanh bên trong khi EMulti đang chạ y. Nh ữ ng bi ểu tượ ng này cung c ấp cho ngườ i dùng thông tin có giá tr ị v ề tr ạ ng thái hi ệ n t ạ i c ủ a E-Multi. N ế u bi ểu tượ ng màu l ụ c, thì t ứ c là thi ế t b ị đang hoạt độ ng. N ế u bi ểu tượ ng có màu xám, thì t ứ c là thi ế t b ị không ho ạt độ ng
